## Method

### Prerocessing

- Case Folding
- Tokenizing
- Filtering
- Word Handling
- Stemming

### Feature Selection

- TF-IDF

### Classification

- Logistic Regression

### Handling Imbalance

- Undersampling
- Oversampling
- SMOTE
- Cost-Sensitive Learning
- Bagging
- Tomek Links
